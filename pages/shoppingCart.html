AngularJS Shopping Cart

Description: Personal freelance web development business 2012-2016

View the live site
View the source code

This project was the result of a small challenge I put myself to. I was curious on how single page applications worked, so I created a shop with three parts:

  • A form to add new products to a "shop"
  • A list that represents products in the shop
  • A subset list of selected items from the shop

The application was built using AngularJS with Twitter Bootstrap for a simple template. The whole project took me about 6 hours. The shop also validates whether a correct value was put in! When testing the app, make sure to know that the information is stored locally, not in a database. The information won't be restored on refresh!
